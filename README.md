## Getting Started

To get a local copy up and running follow these simple example steps.

### Installation

1. Clone the repo
   ```sh
   git clone https://github.com/ks1009/Cycle-Tracker.git
   ```
2. Install NPM packages
   ```sh
   npm install
   ```
3. Run this command
   ```sh
   npm run dev
   ```
